# Getting Started with the OpenPIV-Python Skill

## Overview
This guide shows the quickest way to process a pair of PIV images, visualise the resulting flow field, and start exploring the physics behind the flow. It is aimed at graduate students and researchers who have two consecutive frames from a PIV experiment and want to:
1. Compute the displacement vectors.
2. Visualise velocity magnitude and direction.
3. Ask higher‑level questions about the flow (e.g., identify vortices, calculate circulation, estimate shear layers).

## Prerequisites
- Python 3.10+ (the skill works in an `uv`‑managed environment).
- Two grayscale image files (e.g., `frame1.tif` and `frame2.tif`).
- The **OpenPIV‑Python‑Skill** installed in editable mode:
  ```bash
  uv pip install -e .   # from the repository root
  ```
- The **pypiv** library from Alex Lib (automatically installed as a dependency).

## Step‑by‑Step Workflow
### 1️⃣ Run the CLI
```bash
openpiv-process \
  --image frame1.tif \
  --image frame2.tif \
  --output_dir results \
  --algorithm openpiv_piv \
  --mask dynamic \
  --threads 4 \
  --verbose
```
- `--algorithm` chooses the correlation method (default = `openpiv_piv`).
- `--mask` can be `static` or `dynamic`; `dynamic` automatically masks low‑intensity regions.
- Results are written to `results/params.npz` and a PNG vector field.

### 2️⃣ Load the results in a notebook or script
```python
from openpiv_skill.analyze import PypivAnalyzer

# Load the parameters file generated by the CLI
analyzer = PypivAnalyzer("results/params.npz")

# Plot the velocity field (matplotlib based)
analyzer.plot_vector_field()
```
You’ll see arrows representing displacement vectors overlaid on the original image.

### 3️⃣ Quick quantitative diagnostics
```python
# Basic statistics
print("Mean velocity (pixel/frame):", analyzer.mean_velocity())
print("Signal‑to‑Noise ratio:", analyzer.snr())
```
These numbers give you an immediate sense of the flow magnitude and data quality.

### 4️⃣ Asking deeper questions
The `PypivAnalyzer` class provides a set of helper methods for common fluid‑dynamics queries:
- **Identify vortices** – `analyzer.detect_vortices(min_radius=5)` returns centre coordinates and circulation.
- **Compute shear layer thickness** – `analyzer.shear_layer(threshold=0.2)`.
- **Extract streamlines** – `analyzer.streamlines(step=5)`.

#### Example: Find the strongest vortex
```python
vortices = analyzer.detect_vortices(min_radius=5)
strongest = max(vortices, key=lambda v: v["circulation"])
print(f"Strongest vortex at {strongest['center']} with circulation {strongest['circulation']:.2f}")
```
You can now discuss the physics: why does a vortex appear there? Is it related to a boundary condition? These analyses give you a concrete basis for scientific discussion.

## FAQ
- **What if my images are colour?** Convert them to greyscale before running the CLI: `convert frame1.png -colorspace Gray frame1.tif`.
- **My flow is very fast, and vectors are out of range?** Increase the interrogation window size via `--window_size 64` or enable the synthetic‑aperture mode (`--algorithm synthetic_aperture`).
- **Can I process many frame pairs automatically?** Yes, wrap the CLI in a Python loop or use the `BatchRunner` class (see `tutorials/batch_processing.py`).
